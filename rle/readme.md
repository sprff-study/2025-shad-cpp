# Run-length encoding (RLE)

В этой задаче требуется реализовать алгоритм сжатия без потерь [RLE](https://en.wikipedia.org/wiki/Run-length_encoding). Принцип заключается в замене серий одинаковых символов на количество повторений и сам этот символ. Для упрощения во входной строке не будет символов ['0'-'9'].

### Пример

`AAAABBBCCXYZDDDDEEEFFFB -> A4B3C2XYZD4E3F3B` 

### Примечания

* Обратите внимание, что к символу, который встречается в серии один раз, не приписывается 1.
* Чтобы преобразовать число в строку, возвользуйтесь [std::to_string](https://en.cppreference.com/w/cpp/string/basic_string/to_string).
