# Steal password

Представьте очень секретный сервис, доступ к которому защищён паролем. Чтобы воспользоваться этим сервисом, нужно передать пароль в функцию [`DoSecretStuff`](verify.cpp), где и происходит валидация доступа. Однако, когда аудитория доверенных пользователей сервиса начала расти, потребовалось оперативно сообщать пароль, причём пересылать его напрямую опасно. Посидели-подумали и ввели секретные коды, по которым можно получить пароль через функцию [`GetPassword`](verify.cpp). Реализация этой нехитрой обороны находится в файле `verify.cpp`.

Ваша задача показать, что такая передача пароля отнюдь не безопасна и написать функцию `StealPassword` в файле `steal.h` так, чтобы `DoSecretStuff` не заметил подвоха.

Код собирается компилятором GCC g++-14 без оптимизаций. Локально можно собрать как и любую другую задачу:
```sh
build$ make steal_password
```

Нужно добиться того, чтобы в результате запуска напечаталось `welcome to the secret place...`
