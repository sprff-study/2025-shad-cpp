# Longest subsequence

Вам дана последовательность в виде итераторов и предикат. Необходимо вернуть максимальную по количеству элементов подпоследовательность подряд идущих элементов, где для каждой пары соседних элементов (x, y) предикат pred(x, y) возвращает true. Если таких подпоследовательностей несколько, то вернуть первую, а если их нет - вернуть пару {last, last}.

### Примечания
* Итераторы по крайней мере Forward.
* Если подпоследовательность существует, то в ней как минимум два элемента.
* Использовать `std::ranges` или `std::views` запрещено.
