# Transposition cipher

Transposition cipher - это шифр, который некоторым образом переставляет символы в тексте. Важным здесь является то, что он **переставляет**,  а не заменяет (в отличие от шифра Цезаря).

Например, пусть исходное сообщение "Encrypt me". В нём 10 символов, которые можно представить в виде матрицы `2x5`:
```
'E' 'n' 'c' 'r' 'y'
'p' 't' ' ' 'm' 'e'
```
А теперь просто прочитаем сообщение по столбцам. Получим "Epntc rmye". Выражаясь математически, мы транспонировали матрицу исходного сообщения.

Ваша задача реализовать функцию `Decipher(str)`, которая принимает зашифрованное сообщение и возвращает *варианты* исходного. Так как представление сообщения в виде матрицы сводится к разложению его длины на произведение 2х чисел, а ключ (который задаёт одно из этих чисел) в функцию не передаётся, то образуется неоднозначность. Вам достаточно представить длину сообщения в виде произведения 2х чисел всеми возможными способами и для каждого дешифровать сообщение. Таким образом, исходное сообщение будет одним из элементов [вектора](https://en.cppreference.com/w/cpp/container/vector.html), который возвращает функция `Decipher`.
