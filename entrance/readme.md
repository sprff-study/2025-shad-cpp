# Поступление

Реализуйте функцию [`GetStudents`](entrance.h), которая будет по списку поступающих и университетов определять, кто именно поступит в каждый университет.

Структура университета описана в файле [`university.h`](university.h) и включает название университета и количество свободных мест в нём. У каждого абитуриента есть определенный набранный балл и список предпочтений среди университетов (в порядке убывания). Подробнее см. файл [`student.h`](student.h).

Распределение происходит следующим образом: абитуриенты сортируются по убыванию баллов. При равенстве баллов - по возрастанию даты рождения и, затем, по фамилии и имени. Далее каждый абитуриент по очереди зачисляется в первый университет из его списка, в котором ещё остались места (если таких нет, то не зачисляется никуда).

Функция должна вернуть [`std::map`](https://en.cppreference.com/w/cpp/container/map.html), в котором для каждого университета записаны поступившие в него студенты, **отсортированные по фамилии и имени**. Если в каком-то университете нет поступивших студентов, то он всё равно должен быть в map с пустым списком в качестве значения.

### Примечания
* Используйте библиотечные функции сортировки (например, [`ranges::sort`](https://en.cppreference.com/w/cpp/algorithm/ranges/sort.html)).
* Чтобы сравнить поля структуры как кортежи, воспользуйтесь [std::tie](https://en.cppreference.com/w/cpp/utility/tuple/tie.html).
* У задачи есть приватные тесты (потому что они не очень информативные). Для себя тесты напишите самостоятельно. В качестве примера используйте тот, который уже написан в [`test.cpp`](test.cpp).
* Проверяется только файл [`entrance.h`](entrance.h).
